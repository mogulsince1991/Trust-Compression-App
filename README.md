# Trust Compression App

A minimal Trust Library MVP for turning existing video assets into searchable, shareable trust journeys.

## Core workflows

- Library Manager: review imports, summaries, tags, and content gaps.
- Sales Rep: create prospect briefs, review recommended journeys, and share private links.
- Owner: see useful proof, team usage, and missing content.
- Prospect: watch a calm private journey page.

## Stack

- Next.js App Router
- Supabase
- OpenAI API
- Vercel
